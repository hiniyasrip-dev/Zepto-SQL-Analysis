# Zepto SQL Data Analysis

## Overview

This project is an exploratory data analysis of a Zepto product dataset using MySQL. The objective was to analyze product inventory, pricing, discounts, stock availability, and category-wise performance to gain business insights from retail data.

As a student learning Data Analytics, this project helped me strengthen my SQL skills and understand how data can be used to support business decisions.

---

## Dataset Information




* Total Records: 3,728
* Unique Products: 1,674
* Categories: 14

Dataset includes:

* Product Name
* Category
* MRP
* Discount Percentage
* Selling Price
* Available Quantity
* Product Weight
* Stock Status

Dataset Source:  https://www.kaggle.com/datasets/palvinder2006/zepto-inventory-dataset?resource=download

## Key Insights

* Analyzed 3,728 product records across 14 categories.
* Found 1,674 unique products in the dataset.
* Cooking Essentials had the highest inventory value.
* Biscuits category showed the highest stockout percentage (28.57%).
* Dukes Waffy products offered the highest discount (51%).
* Borges Extra Light Olive Oil provided the highest customer savings.
* Several products appeared multiple times, indicating repeated inventory records.
